# user_active_time
Монитор активности пользователя Windows.
Работает без прав администратора.

**Что отслеживается:**
- `LOGON` — вход в систему
- `LOGOFF` — выход из системы
- `LOCK` — блокировка экрана (Win+L)
- `UNLOCK` — разблокировка экрана

**Как использовать:**

1. Запусти: `python main.py`
2. Для остановки: закрой виджет (✕) или `Ctrl+C`

**Где хранятся данные:**

Всё пишется в папку `C:\Users\<пользователь>\active_time\`:
- `userName_dd.mm.yyyy.txt` — отчёт за день (активное время, сессии, лог событий)
- `state.json` — внутреннее состояние программы (не редактировать)

**Виджет активности:**

При запуске на рабочем столе появляется виджет с текущей статистикой:
- Активное время за день
- Количество сессий
- Процент активности

Цвет фона виджета меняется в зависимости от процента активности:
- Зелёный — выше рекомендуемого порога (по умолчанию 80%)
- Жёлтый — между минимальным и рекомендуемым порогом
- Красный — ниже минимального порога (по умолчанию 70%)

Виджет можно перетаскивать за заголовок. Закрытие виджета завершает приложение.

**Настройки:**

Файл `config.py`:
- `MAX_WORK_HOURS` — максимальное рабочее время в часах (по умолчанию 8)
- `RECOMMENDED_ACTIVITY_THRESHOLD` — рекомендуемый порог активности, % (по умолчанию 80)
- `MIN_ACTIVITY_THRESHOLD` — минимально допустимый порог, % (по умолчанию 70)
- `WIDGET_SHOW_ACTIVE_TIME` — показывать активное время в виджете (True/False)
- `WIDGET_SHOW_SESSION_COUNT` — показывать количество сессий (True/False)
- `WIDGET_SHOW_ACTIVITY_PERCENT` — показывать процент активности (True/False)

Если все три опции отображения выключены (False), виджет не создаётся.

**Автозапуск без админки:**

Добавь ярлык в автозагрузку пользователя:
```
Win+R → shell:startup → создай ярлык на pythonw.exe main.py
```

**Структура проекта:**
- `main.py` — точка входа
- `config.py` — настройки приложения
- `constants.py` — константы Windows API
- `utility.py` — утилиты (форматирование дат/времени)
- `report.py` — формирование отчёта
- `modules/session_monitor.py` — мониторинг сессий Windows
- `modules/widget.py` — виджет активности на рабочем столе
